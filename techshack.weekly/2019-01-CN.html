# EnqueueZero Techshack 2019-01

大家好，新年快乐！

不知不觉，继 Techshack 第 0000 期 (opens new window) 发出已经整整一周年了。2018 年 Techshack Bi-Weekly Weekly 一共只发出了 25 期, 可见我是个无可救药的懒散家伙！在新的一年里，我希望可以 洗心革面 再接再厉，做的更勤快些，整理出更有用的素材!

首先，由于来自不愿意透露姓名 EnqueueZero 金主爸爸 @雨忆 (opens new window) 的灵魂拷问：

techshack 已经停刊了吗

我决定，从 2019 年开始，Techshack 不仅正式合并到我的另一个项目 EnqueueZero 中, 还会继续有中文导读的更新。

最后，言归正传，下面是 2019-01 的阅读推荐列表：

  • Kubernetes In a Nutshell (opens new window), 对 Kubernetes 里面的重要概念做了极简入门。如果你确定你都了解了这些概念，不如自己也整理一遍，确认你也可以向人说清楚（费曼学习法）。
  • How Discord Stores Billions of Messages (opens new window), 对于聊天应用如何存储十亿级别数据量的聊天消息如何存储做出了简单介绍，包括数据存储，分库，数据库选型等等。
  • Everything You Need To Know About Networking On AWS (opens new window), 简单介绍了 AWS VPC 子网的重要概念。美中不足的是还有不少概念没说清，稍后我会有相对应的文章发布。
  • An Idiot’s guide to Support vector machines (SVMs) (opens new window).
  • How do you document a tech project with comics? (opens new window), 如何使用漫画介绍技术概念？
  • High-Reliability Infrastructure Migrations (opens new window), 来自 ^ 同一个作者 Julia Evans 在 KubeCons 上的演讲，关于如何在 Kubernetes 集群上做 Infrastructure 的变更。
  • Introducing Pipelines to Airbnb’s Deployment Process (opens new window), 一个系列文章。
  • Learn eBPF Tracing: Tutorial and Examples (opens new window), 来自 Brendan Gregg's 关于 ebpf, bcc, bpftrace 的学习资源汇总。
  • Docker Internals (opens new window), 罗列了 Docker 的内部实现的重要概念
  • Everything Curl (opens new window), 来自 Curl 作者写的一本小书，推荐阅读。
  • GraphQL Server Design @ Medium (opens new window), 介绍了 Medium 如何将 GraphQL 引入作为 API 层。

Kia ora! I am the creator of Enqueue Zero (https://enqueuezero.com), a site that explains code principles to Developers/DevOps/Sysadmins. This site is updated under a single man team since 2018. The `EnqueueZero Techshack` series is comprised of a bunch of interesting posts I wrote or found on the Internet. More importantly, I hope these posts would please you.

Please follow Twitter account @enqueuezero for the updates. Happy exploring!
